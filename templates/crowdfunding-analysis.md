## 募资定价策略

### 早鸟定价模型

| 档位 | 数量 | 折扣 | 策略目的 |
| --- | --- | --- | --- |
| 超级早鸟 | 50-100 | 40%-50% off | 快速积累首批支持者，制造紧迫感 |
| 早鸟 | 100-200 | 20%-30% off | 吸引价格敏感型用户 |
| 标准版 | 不限 | 原价 | 正常售价 |
| 豪华版 | 不限 | 10%-20% premium | 提供增值服务或配件 |

### 定价原则

1. **成本覆盖**：早鸟价也要覆盖 BOM 成本和平台费用
2. **价格梯度**：至少 3 个价格档位，形成选择压力
3. **稀缺感**：限量早鸟制造紧迫感
4. **价值感知**：每个档位要有明确的价值差异
5. **比价空间**：最终零售价要高于早鸟价，让支持者感觉划算

## 履约风险评估

### 风险清单

| 风险类型 | 风险描述 | 风险等级 | 应对措施 |
| --- | --- | --- | --- |
| 供应链延迟 | 元器件缺货、产能不足 | 高 | 多供应商备选、安全库存 |
| 质量问题 | 产品质量不达标 | 高 | 严格质检、第三方检测 |
| 物流延误 | 发货延迟、运输损坏 | 中 | 提前备货、选择可靠物流 |
| 合规问题 | 认证不全、法规限制 | 中 | 提前申请认证、了解法规 |
| 资金管理 | 资金不足、现金流断裂 | 高 | 预留资金、分期生产 |
| 团队能力 | 交付经验不足 | 中 | 找有经验的合作伙伴 |

### 履约计划模板

| 阶段 | 时间 | 任务 | 责任人 | 风险控制 |
| --- | --- | --- | --- | --- |

| 供应链确认 | 募资前 | 确认供应商和交期 | | 签订供货合同 |
| 量产准备 | 募资中 | 准备模具、原材料 | | 预付定金锁定产能 |
| 生产制造 | 募资后1-2月 | 批量生产 | | 每周跟进生产进度 |
| 质量检测 | 生产完成后 | 产品检测和认证 | | 第三方质检 |
| 包装发货 | 检测通过后 | 包装和物流安排 | | 分批发货，优先早鸟 |
| 售后支持 | 发货后 | 客服和售后处理 | | 准备常见问题FAQ |

## 募资文案要素

### 标题要求

| 要素 | 说明 | 示例 |
| --- | --- | --- |
| 产品名 | 简短易记 | "TITANSHIELD" |
| 核心卖点 | 一句话讲清价值 | "防爆固態行動電源" |
| 差异化 | 与竞品的区别 | "全球首款" |

### 首图要求

| 要素 | 说明 | 标准 |
| --- | --- | --- |
| 产品展示 | 清晰展示产品 | 高分辨率产品图 |
| 使用场景 | 展示使用场景 | 真实场景图 |
| 核心价值 | 突出核心价值 | 对比图、效果演示 |

### 视频要求

| 要素 | 时长 | 说明 |
| --- | --- | --- |
| 开头钩子 | 0-10秒 | 3秒内抓住注意力 |
| 问题呈现 | 10-30秒 | 展示用户痛点 |
| 产品介绍 | 30-60秒 | 展示产品功能和效果 |
| 使用演示 | 60-90秒 | 真实使用场景演示 |
| 价值总结 | 90-120秒 | 总结核心价值和早鸟优惠 |

### 正文结构

```
1. 痛点描述（为什么需要这个产品）
2. 产品介绍（这个产品是什么）
3. 核心功能（产品能做什么）
4. 使用场景（什么时候使用）
5. 技术参数（专业数据支撑）
6. 团队介绍（为什么我们能做好）
7. 早鸟优惠（为什么现在支持）
8. 进度计划（什么时候交付）
9. 常见问题（Q&A）
```

## 平台审核要点

### 嘖嘖审核要点

| 审核项 | 要求 | 建议 |
| --- | --- | --- |
| 产品原型 | 必须有真实原型 | 准备实物照片和视频 |
