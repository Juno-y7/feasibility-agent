# 数据指标框架

## 北极星指标

### 定义

北极星指标（North Star Metric）是一个能够衡量产品核心价值是否被用户实现的唯一指标，它回答了"我们是否正在为用户创造价值"这个问题。

### 品类特定北极星指标

| 品类 | 北极星指标示例 | 说明 |
| --- | --- | --- |
| 📦 硬件/消费品 | 活跃用户数 × 客单价 × 复购率 | 衡量产品被使用和产生收入的能力 |
| 🛠️ SaaS/效率工具 | MRR（月经常性收入） | 衡量订阅业务的健康度 |
| 🤝 服务/咨询 | 客户生命周期价值（LTV） | 衡量服务的长期价值 |
| 🧠 情感/内容/社群 | DAU（日活跃用户数） | 衡量社区的活跃度和粘性 |
| 🏢 B2B/企业级 | 年度合同价值（ACV） | 衡量企业客户的价值 |
| 🌱 平台/双边市场 | GMV（商品交易总额） | 衡量平台的交易规模 |

### 北极星指标公式

```
北极星指标 = 核心用户数 × 核心行为频次 × 核心行为价值
```

## AARRR 漏斗模型

### 获取（Acquisition）

| 指标 | 定义 | 目标值 | 当前状态 |
| --- | --- | --- | --- |
| 渠道曝光量 | 产品被展示的次数 |  | |
| 渠道点击量 | 用户点击进入产品的次数 |  | |
| 渠道转化率 | 点击→注册/下载的转化率 |  | |
| CAC（客户获取成本） | 获取一个付费客户的成本 |  | |
| 渠道ROI | 渠道投入产出比 |  | |

### 激活（Activation）

| 指标 | 定义 | 目标值 | 当前状态 |
| --- | --- | --- | --- |
| 注册/下载量 | 完成注册或下载的用户数 |  | |
| 首单转化率 | 注册→首次使用/购买的转化率 |  | |
| 核心功能使用率 | 使用核心功能的用户占比 |  | |
| 激活时长 | 从注册到首次使用核心功能的时间 |  | |
| 激活成功率 | 完成激活流程的用户占比 |  | |

### 留存（Retention）

| 指标 | 定义 | 目标值 | 当前状态 |
| --- | --- | --- | --- |
| D1留存 | 次日留存率 |  | |
| D7留存 | 7日留存率 |  | |
| D30留存 | 30日留存率 |  | |
| 周留存率 | 连续两周活跃的用户占比 |  | |
| 月留存率 | 连续两月活跃的用户占比 |  | |
| 用户生命周期 | 用户从首次使用到流失的平均时长 |  | |

### 变现（Revenue）

| 指标 | 定义 | 目标值 | 当前状态 |
| --- | --- | --- | --- |
| ARPU（单用户平均收入） | 总收入/用户数 |  | |
| ARPPU（付费用户平均收入） | 总收入/付费用户数 |  | |
| 付费转化率 | 活跃用户→付费用户的转化率 |  | |
| LTV（用户生命周期价值） | 单个用户在生命周期内贡献的总收入 |  | |
| LTV/CAC 比值 | 用户生命周期价值与客户获取成本的比值 | >=3 | |

### 推荐（Referral）

| 指标 | 定义 | 目标值 | 当前状态 |
| --- | --- | --- | --- |
| NPS（净推荐值） | 用户推荐意愿的度量 |  | |
| 推荐转化率 | 邀请→注册/购买的转化率 |  | |
| 邀请率 | 发出邀请的用户占活跃用户的比例 |  | |
| 病毒系数 | 每个用户平均带来的新用户数 | >=1 | |
