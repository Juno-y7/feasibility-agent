# 产品可行性分析智能体

## 产品概述

**产品可行性分析智能体**是一款专门帮助创业者、产品经理和投资人判断产品方案可行性的AI分析助手。只需输入项目资料，智能体就能自动完成从品类识别、竞品对比到可行性评分的全流程分析，输出结构化的可行性报告。

## 核心价值

### 为什么需要这个智能体？

在产品立项阶段，大多数人依赖直觉和经验判断，缺乏系统性的分析框架。这个智能体解决了以下痛点：

- **拍脑袋决策**：提供10维度加权评分模型，让决策有数据支撑
- **信息不完整**：自动从零散资料中抽取关键信息，不要求用户填表
- **缺乏参照**：内置真实案例库，找到相似项目作为参照系
- **盲目乐观**：反方审查机制自动识别乐观假设和证据不足

### 核心优势

1. **可复用的分析框架**：换任何产品，只要把资料丢进去，就能自动产出报告
2. **科学的评分标准**：10个维度加权打分，不再是拍脑袋决策
3. **真实案例库**：包含嘖嘖、Kickstarter等平台的真实项目数据
4. **反方审查机制**：自动揪出报告里的乐观假设和证据不足

## 功能介绍

### 三步分析流程

```
项目资料 
     ↓ 
第一步：识别项目画像 
     → 这是什么品类？ 
     → 它解决什么问题？ 
     → 它像谁？ 
     ↓ 
第二步：去募资平台验证 
     → 搜索同类项目 
     → 对比关键指标 
     → 找出成功/失败差异点 
     ↓ 
第三步：回推判断 
     → 在生态中的位置？ 
     → 机会在哪？风险在哪？ 
```

### 详细功能

#### 1. 品类识别
- 自动判断产品品类：AI硬件、软件工具、消费电子、生活方式、文创设计、游戏娱乐、机器人、智能穿戴、智能家居等
- 根据品类自动切换评分权重和检查项

#### 2. 痛点分析
- 分析用户核心痛点是否明确、具体、可感知
- 判断痛点强度（高频刚需 vs 低频可选）
- 分析现有解决方案的不足

#### 3. 竞品对比
- 从案例库中找出最相似的3-5个参照项目
- 对比募集金额、达成率、支持者、价格区间等关键指标
- 分析成功/失败项目的差异点

#### 4. 可行性评分
- 10维度加权评分模型，满分100分
- 维度包括：痛点强度、目标用户清晰度、竞争壁垒、产品可演示性、技术可行性、合规与信任、商业模式、市场推广可行性、团队匹配度、证据完整度
- 分数解释：80-100分建议推进，65-79分需补强短板，50-64分需重新定义，0-49分暂不建议投入

#### 5. 风险评估
- 识别竞争风险、技术风险、履约风险、市场风险、合规风险
- 评估风险等级，提供应对建议

#### 6. 行动建议
- 7天验证任务：快速验证关键假设
- 30天推进计划：详细的推进路线图
- 关键短板补强：明确需要优先解决的问题

## 适用场景

- **产品立项评估**：判断新产品想法是否值得投入
- **募资预热准备**：评估项目是否适合上嘖嘖、Kickstarter等平台
- **投资决策辅助**：帮助投资人筛选优质项目
- **团队内部讨论**：提供客观的分析框架，促进团队共识

## 使用方式

### 输入
- 产品介绍文档
- 官网链接或草稿
- 竞品信息
- 团队背景
- 任何相关资料

### 输出
完整的可行性分析报告，包含：
1. 项目画像（品类、痛点、参照项目）
2. 募资平台验证（竞品对比、成功/失败特征）
3. 回推判断（生态位置、机会、风险）
4. 可行性评分（10维度加权打分）
5. 行动建议（7天验证任务、30天推进计划）
6. 沟通话术（汇报用）

## 技术规格

- **支持平台**：豆包、讯飞星火、GPT-4等主流AI平台
- **上下文要求**：最低8K，推荐16K+
- **输出格式**：结构化Markdown报告
- **语言支持**：中文

## 隐私说明

- 用户输入的项目资料仅用于当前分析，不会存储或用于其他用途
- 分析过程在AI模型中完成，数据处理符合平台隐私政策
- 如需长期保存分析结果，用户可自行导出报告
