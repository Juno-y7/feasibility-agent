## 二、核心功能

### 2.1 功能列表
- [ ] 品类识别（自动判断产品品类）
- [ ] 痛点分析（分析用户需求强度）
- [ ] 竞品对比（从案例库中匹配相似项目）
- [ ] 可行性评分（10维度加权评分）
- [ ] 风险评估（识别潜在风险和应对建议）
- [ ] 行动建议（7天验证任务 + 30天推进计划）

### 2.2 特色亮点
- [ ] 三步分析流程：识别项目画像 → 募资平台验证 → 回推判断
- [ ] 内置真实案例库：包含嘖嘖、Kickstarter等平台的真实项目数据
- [ ] 反方审查机制：自动识别乐观假设和证据不足
- [ ] 可执行输出：不只是分析，更提供具体行动方案
